프로젝트 이름 : 나의 약 지갑

<목적>
- 응급상황에서 의료진들이 빠르게 내가 복용중인 약을 확인할 수 있도록 하여 효율적이고 신속한 대처를 할 수 있다.
- 본인 또는 보호자가 직접 정보를 입력·수정할 수 있어야 하며, 별도 설치나 로그인 없이 스마트폰 브라우저에서 바로 열어 보여줄 수 있어야 한다.

<기능>
- 응급 화면(기본 화면): 이름, 생년월일, 혈액형, 알레르기, 지병(기저질환), 복용 중인 약 목록, 보호자/비상연락처를 큰 글씨로 한눈에 표시
- 알레르기·주의사항은 빨간색으로 강조해 가장 먼저 눈에 띄도록 표시
- 편집 화면: 위 정보를 직접 추가·수정·삭제 가능 (약은 여러 개 등록 가능: 약 이름, 용량, 복용 시간/주기, 복용 목적)
- 입력한 정보는 브라우저(로컬 저장소)에 자동 저장되어 앱을 다시 열어도 유지됨
- 큰 글씨 모드 및 고대비 색상으로 노약자·시력이 안 좋은 사용자도 쉽게 인식 가능
- 별도 서버·설치 없이 HTML 파일 하나로 실행 (인터넷 연결 없이도 동작)
- 잠금화면 알림으로 응급정보 요약 표시: 홈 화면에 앱 추가(PWA) 후 알림 권한을 허용하면, 이름/혈액형/알레르기/복용약/보호자 연락처 요약을 잠금화면 알림으로 띄워 잠금 해제 없이도 확인 가능. 정보를 저장할 때마다 알림 내용도 자동 갱신됨
  ※ 단, 이 기능은 서비스 워커를 사용하므로 HTML 파일을 직접 열 때는 동작하지 않고, localhost 또는 HTTPS로 호스팅된 상태에서만 동작함 (다른 기능은 파일을 직접 열어도 정상 동작)
