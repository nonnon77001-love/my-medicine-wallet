프로젝트 이름 : 약쏙 (구 이름: 나의 약 지갑)

<목적>
- 응급상황에서 의료진들이 빠르게 내가 복용중인 약을 확인할 수 있도록 하여 효율적이고 신속한 대처를 할 수 있다.
- 본인 또는 보호자가 직접 정보를 입력·수정할 수 있어야 하며, 별도 설치나 로그인 없이 스마트폰 브라우저에서 바로 열어 보여줄 수 있어야 한다.

<기능>
- 응급 화면(기본 화면): 이름, 생년월일, 혈액형, 알레르기, 지병(기저질환), 복용 중인 약 목록, 보호자/비상연락처를 큰 글씨로 한눈에 표시
- 알레르기·주의사항은 기본정보/지병과 동일한 흰색 카드로 표시 (박스 자체는 색으로 구분하지 않고, 위험도 배지와 종합 위험도 배너로만 경중을 표시)
- 편집 화면: 위 정보를 직접 추가·수정·삭제 가능 (약은 여러 개 등록 가능: 약 이름, 용량, 복용 시간/주기, 복용 목적)
- 입력한 정보는 브라우저(로컬 저장소)에 자동 저장되어 앱을 다시 열어도 유지됨
- 큰 글씨 모드 및 고대비 색상으로 노약자·시력이 안 좋은 사용자도 쉽게 인식 가능
- 별도 서버·설치 없이 HTML 파일 하나로 실행 (인터넷 연결 없이도 동작)
- 잠금화면 알림으로 응급정보 요약 표시: 홈 화면에 앱 추가(PWA) 후 알림 권한을 허용하면, 이름/혈액형/알레르기/복용약/보호자 연락처 요약을 잠금화면 알림으로 띄워 잠금 해제 없이도 확인 가능. 정보를 저장할 때마다 알림 내용도 자동 갱신됨
  ※ 단, 이 기능은 서비스 워커를 사용하므로 HTML 파일을 직접 열 때는 동작하지 않고, localhost 또는 HTTPS로 호스팅된 상태에서만 동작함 (다른 기능은 파일을 직접 열어도 정상 동작)
- 위험정도 구분: 알레르기에 위험도(경미/보통/심각)를 지정하면 색상 배지로 표시되고, 약물별로 "중요(필수) 약물"로 표시하면 목록 맨 위로 정렬되고 별도 강조됨. 이 정보들을 종합해 화면 최상단에 "종합 위험도(정상/주의/위험)" 배너를 자동으로 보여줌
- 자동 요약: 입력된 정보를 바탕으로 이름·혈액형·알레르기·지병·복용약·보호자 연락처를 한 줄로 압축한 요약 문구를 자동 생성해 상단에 표시 (외부 AI API를 호출하지 않는 규칙 기반 로직 — 공개 저장소에 API 키를 노출하지 않기 위함이자, 인터넷 연결 없이도 동작하게 하기 위함)
- 마지막 업데이트 날짜/시간을 응급 화면 상단에 표시해 정보의 최신성을 확인 가능
- 정보 공유 QR코드: "정보 공유" 버튼을 누르면 현재 응급정보가 담긴 링크를 QR코드로 보여줌. 보호자가 폰 카메라로 스캔하면 같은 정보를 읽기 전용으로 바로 확인 가능하고, 원하면 "내 기기에 저장" 버튼으로 자신의 기기에도 저장할 수 있음
  ※ 정보는 서버에 저장되지 않고 링크 자체(URL 뒷부분)에 암호화 없이 포함되므로, 신뢰할 수 있는 사람에게만 공유해야 함
- 브라우저 탭/즐겨찾기/홈 화면 아이콘(파비콘)도 새 로고로 표시되도록 설정

<추후 고려>
- 실제 생성형 AI 요약이 필요할 경우: 사용자가 자신의 API 키를 직접 입력해 브라우저에서 호출하는 방식(BYOK)을 고려할 수 있으나, 키가 클라이언트에 노출되는 구조적 한계가 있어 신중한 결정 필요
